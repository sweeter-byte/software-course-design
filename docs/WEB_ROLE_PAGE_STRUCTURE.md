# Web 端角色页面结构设计

更新日期：2026-05-20

本文档记录 Web 端按角色组织的页面结构。当前已确认学生端结构；教师端和教务员端后续继续补充在同一文档中。

## 1. 总体原则

Web 端不再直接按需求文档中的功能模块或数据库类拆分左侧导航。真实业务链路应围绕教学上下文组织：

```text
课程 -> 作业 -> 提交/批改 -> 作业问题或反馈 -> 教师答疑回复
```

因此，左侧导航只进入大的业务空间；具体操作应在课程、作业、提交这些上下文内完成。

反馈分为两类：

1. 作业反馈：绑定某次作业和批改结果，是学生围绕具体提交、分数、教师评语继续沟通的主反馈流。
2. 课程整体反馈：绑定课程反馈任务，不绑定某次作业。课程结束后自动出现，教师也可以在课程进行中手动触发。

课程整体反馈采用任务制，不是长期开放意见箱。学生提交课程整体反馈时始终可以选择匿名；匿名仅对教师匿名，系统和教务员保留审计能力。

## 2. 学生端

### 2.1 左侧导航

学生端左侧导航建议保留以下入口：

```text
工作台
我的课程
我的作业
课程反馈任务
消息
账号维护
```

入口职责：

| 入口 | 职责 |
| --- | --- |
| 工作台 | 展示学生当前最需要处理的学习待办，例如待提交作业、已批改作业提醒、待填写课程整体反馈任务。 |
| 我的课程 | 查看已加入课程和可加入课程；进入某门课程后进入课程工作区。 |
| 我的作业 | 跨课程汇总所有作业状态，点击作业后进入对应课程工作区的作业详情。 |
| 课程反馈任务 | 汇总所有课程级反馈任务，包括待填写、已提交、已过期任务。只处理课程整体反馈，不处理作业反馈。 |
| 消息 | 汇总教师回复、系统提醒、课程反馈任务提醒等通知，不作为发起反馈的主要入口。 |
| 账号维护 | 个人资料、手机号、密码、安全操作。 |

### 2.2 课程工作区

学生从“我的课程”进入某门课程后，进入课程工作区。

课程工作区顶部固定展示课程上下文：

```text
课程名称 / 授课教师 / 学期 / 上课时间地点 / 课程状态 / 当前学习进度
```

课程工作区使用以下 Tab：

```text
课程概览 / 作业 / 作业反馈 / 课程整体反馈 / 资料与通知
```

### 2.3 课程概览

课程概览用于帮助学生确认自己当前所在课程，不承担密集操作。

主要内容：

- 课程简介
- 授课教师
- 学期
- 上课时间与地点
- 课程状态
- 课程人数或已加入状态
- 最近待办摘要

### 2.4 作业

作业 Tab 是学生端主操作区。

作业列表展示每次作业的状态：

- 未开始
- 进行中
- 已提交
- 已批改
- 已截止

点击某次作业后，在同一页面完成：

- 查看作业说明、要求、开始时间、截止时间
- 提交答案
- 在允许条件下修改答案
- 查看提交状态
- 查看分数
- 查看教师批改评语

作业详情页必须天然带有以下上下文：

```text
courseId / assignmentId / submissionId
```

### 2.5 作业反馈

作业反馈不再作为左侧独立主入口，而是课程工作区内的上下文功能。

作业反馈 Tab 按“已批改作业”列出可反馈对象。每条记录展示：

- 作业名称
- 分数
- 教师评语
- 是否已有问题或反馈
- 教师是否已回复

学生点击某条作业反馈对象后，进入该作业的反馈线程。

发起作业反馈的推荐条件：

- 学生必须已有提交记录。
- 推荐在作业已批改后才允许发起问题或反馈。
- 反馈内容必须绑定到具体 `courseId / assignmentId / submissionId`。

作业反馈线程展示：

- 学生问题或反馈内容
- 教师回复
- 回复时间
- 当前处理状态

### 2.6 课程整体反馈

课程整体反馈只处理课程级反馈任务。

课程整体反馈任务来源：

1. 自动触发：课程状态变为“已结课”，或课程结束日期到达后，系统自动生成课程整体反馈任务。
2. 教师手动触发：课程进行中，教师可在课程工作区手动发起课程整体反馈任务。

课程整体反馈任务字段建议：

- 反馈标题
- 反馈说明
- 开始时间
- 截止时间
- 任务状态
- 是否已提交

学生提交课程整体反馈时：

- 始终允许勾选“匿名提交”。
- 匿名仅对教师匿名。
- 系统和教务员仍保留真实提交人，用于审计。
- 学生自己查看历史反馈时，应能看到自己提交过该反馈，并能看到是否匿名提交。

教师端展示匿名反馈时：

```text
匿名学生
```

不得展示学生姓名、学号、班级、手机号等身份信息。

教务员端审计时可以查看真实提交人。

### 2.7 资料与通知

该 Tab 作为课程附属信息区，短期可以保持轻量。

可展示：

- 课程说明
- 教师补充通知
- 课程安排变更
- 后续可扩展的课件或资料入口
